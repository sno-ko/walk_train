# walk_train
### 尺取り虫／四足歩行動物の遺伝的アルゴリズムを用いた前進のシミュレーション
（開発：2019年12月~2020年3月）

**実行方法：**

尺取虫の場合
```
cd inchworm
python worm.py
```

二足歩行動物の場合
```
cd four_legs
python four_legs.py
```

**Pythonバージョン：** 3.7.4 

**必要ライブラリ：** Box2D，Numpy
